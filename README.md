# How to build

## Init && Install Modules

Using CNPM (For Chinese)
```
cnpm i
```
Using PNPM
```
pnpm i
```

### Hot-Reload for delevop
```
npm run electron:serve
```

### Build install pack for users
```
npm run electron:build
```
